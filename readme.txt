The WP SMART SMS Plugin allows WordPress Administrators and Viewer to send SMS to their any Mobile users.Specially For India, Pakistan, Bangladesh.

== Description ==
      You May SMS to any Mobile number through the <i>Send SMS</i> Button in Admin Menu.<br>
<br>
No need for configuration of your WP Smart sms settings.Enter your username, password,message and Mobile number through the <i>Settings<Send SMS</i> tab.</br>

== Installation ==

1. Upload zip file to the plugins tab.
2. Activate Plugin and Create a Menu of wpsmart-sms plugins. 
3. See menu in admin of wordpress and click send option.
